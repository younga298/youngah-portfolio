HTML

웹페이지의 구조에 대한 이해가 높고, 시맨틱 태그를 활용하여 페이지 구조를 명확하게 배치할 수 있습니다. CSS의 클래스와 ID를 활용하여 재사용할 수 있는 코드를 작성할 수 있습니다. 이러한 방법들로 작업 속도를 줄일 수 있습니다.

CSS/SCSS

Media Query를 활용하여 반응형 웹디자인을 구현하고, Flexbox와 CSS 그리드를 사용하여 요소의 배치를 깔끔하게 할 수 있습니다. 애니메이션을 사용하여 웹 페이지에 동적인 효과를 추가 할 수 있으며, SCSS를 사용하여 변수, 믹스인, 네스팅 등의 기능을 활용하여 코드의 재사용성에 대해 잘 다룹니다.

JAVASCRIPT

변수를 선언하고 데이터를 할당하며 조건문과 반복문을 수행할 수 있습니다. jquery 기초를 이해하고 있으며 DOM 조작이나 애니메이션, Ajax 기능을 수행할 수 있습니다.
